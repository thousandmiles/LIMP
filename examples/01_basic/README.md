# Basic LIMP Usage

LIMP message construction and parsing (no network).

## Examples

### simple_request.cpp
Build REQUEST messages with different payload types.

**Run**:
```bash
./build/examples/01_basic/simple_request
```

### simple_response.cpp
Build RESPONSE messages with tag quality, timestamps, analog values, etc.

**Run**:
```bash
./build/examples/01_basic/simple_response
```
